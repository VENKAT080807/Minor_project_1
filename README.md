WhatsApp Chat Analyzer
Overview

This is a Python-based project that analyzes exported WhatsApp chat files and provides useful insights such as message statistics, participant activity, media sharing, and deleted messages.

Features
Total number of messages
Most active participant
Media messages count
Deleted messages analysis
Chat statistics
Data visualization
Text preprocessing using Regular Expressions
Technologies Used
Python
Pandas
NumPy
Matplotlib
Regular Expressions (Regex)
Jupyter Notebook
How to Use
Export a WhatsApp chat without media.
Place the exported .txt file in the project folder.
Open the Jupyter Notebook.
Run all the cells.
View the generated analysis and visualizations.
