# 📊 WhatsApp Chat Analyzer

A Python-based project that analyzes exported WhatsApp chat files and generates meaningful insights such as message statistics, participant activity, media sharing, and deleted messages.

## ✨ Features

- Analyze total messages
- Count media messages
- Analyze deleted messages
- Identify the most active participants
- Generate chat statistics
- Data preprocessing using Regular Expressions
- Visualize chat insights

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Regular Expressions (Regex)
- Jupyter Notebook

## 📂 Project Structure

```
WhatsApp-Chat-Analyzer/
│── Mini_Project_1.ipynb
│── README.md
│── requirements.txt
│── sample_chat.txt
```

## 🚀 How to Run

1. Clone this repository.
2. Install the required libraries.
3. Export a WhatsApp chat without media.
4. Place the exported `.txt` file in the project folder.
5. Open the Jupyter Notebook and run all cells.

## 📈 Output

The project provides insights such as:

- Active Users

- Deleted Messages
- Chat Statistics
